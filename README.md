# BMI Calculator

A simple and beginner-friendly BMI Calculator built using Python and Streamlit.

The application allows users to enter their weight and height and instantly calculates their Body Mass Index (BMI).

## Features

* Enter weight in kilograms
* Enter height in centimeters
* Automatically convert height into meters
* Calculate BMI instantly
* Simple and interactive Streamlit interface

## BMI Formula

BMI is calculated using the following formula:

```text
BMI = Weight (kg) / Height² (m)
```

* Streamlit

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/BMI-Calculator.git
```

### 2. Open the project folder

```bash
cd BMI-Calculator
```

### 3. Install the required package

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```

The application will open in your web browser.

## Requirements

Add the following line inside `requirements.txt`:

```text
streamlit
```

## Example

Enter:

```text
Weight: 70 kg
Height: 175 cm
```

Result:

```text
BMI: 22.86
```
